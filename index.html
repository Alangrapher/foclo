Code Review — PR #142
《Alangrapher 计时准则》
The Alangrapher's Code of Timing
"时间不会说谎。"
"Time tells no lie."
规则 1
不存在完美的记录。
Do not chase a flawless log.
规则 2
虚构记录不会创造时间。
Never summon time with fabricated entries.
规则 3
做时间的战友而非俘虏。
Walk with time as a comrade, not a captive.
规则 4
时间不代表成绩。
Don't let time fool you into pride.
规则 5
为每一个不必记录的日子干杯。
Toast to the days you never had to record.
Alan Zhu
